アメリカで発行されている季刊誌、The Perl Journal Japanの日本版を
つくろう、という話があります。（以前にもお知らせしましたが）。

このたび舌逅跂闕のMailing Listを立ち上げましたので、参加していただける
方は購読お願いします。

TPJ-JP Mailing List
	購読の仕方
		宛舌逅跂闕:	majordomo@example.com
		本文:	subscribe tpj-jp
			end
		というメールを送ってください。
	MLのアドレスは tpj-jp@example.com

MLでは日本語、英語両方の投稿が予想されます。

(なおTLUG (Tokyo Linux Users Group)のサーバーを3/14に立ち上げました。
Turbo Linuxで動いています。現在間借り舌逅跂闕のPHTのサブドメインになってい
ますが、ドメイン取得(tlug.gr.jp)で動いていますので将来変更になります。
その時はまたご案内します。)

TPJ-JPの現在までの動きについては下記のWeb Pageをご参照ください。
	http://www.twics.com/~tlug/projects/tpj-jp/index.html

当初はWeb上で出版/公開しようという予定です。まだ纔瘢韭絎竢蹐任垢△修
レイアウト案が下記にあります。
	http://www.digital-mines.com/tpj_jp/

TLUGだけでなくLinux/BSD/Perlの愛好者の方含めて広くご参加いただきた
いと考えております。よろしくお願いします。



==

Some TLUG(Tokyo Linux Users Group) members have initiated the
discussion on publishing Japanese version of "The Perl Journal",
a quarterly magazine of all aspects of Perl.

Those who are interested in participating into this project,
please subscribe to the new Mailing List "TPJ-JP".  To subscribe,
send the following e-mail:
	To:	majordomo@example.com
	Contents: subscribe tpj-jp
		 end

Mailing List address is
	tpj-jp@example.com

In the Mailing List, both messages in Japanese and English
are expected.

(TLUG set up its own server on 14 March, using Turbo Linux.
The name now is a subdomain of PHT, who generously offered its
space and bandwidth.  TLUG is applying for its own domain name
tlug.gr.jp, upon setup the addresses will change - to be informed.)

The progress of TPJ-JP so far can be seen at:
	http://www.twics.com/~tlug/projects/tpj-jp/index.html

A draft web page for publicizing The Perl Journal Japan (contents
in English now) is at:
	http://www.digital-mines.com/tpj_jp/

Not only from TLUG, we invite people from broad area, including
(and not limited to) Linux, BSD, Perl lovers.
