I am looking at some of the setting files for uum.  There is one file
called a Conversion Key File and another file called a Key Bind file. 
They look like they do very similar things to me.  Does anyone know what
they do differently.

at the same time I can find the following the Key Convert file:

Aren't these two doing the same thing?
