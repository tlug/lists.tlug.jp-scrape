I'm using GNU Emacs20.2.  However, I think you still have to kind of 
patch it to get Canna to work properly with it.  I think that XEmacs20.x
is more straightforward.   You can also enter into extension 拡張機能
from kinput2.  For example, if you are on the command line, press Shift-Space,
then when the ［あ］ comes up, type the "Home" key on keyboard.  If you have
a "Help" key, that might work.  I don't have a help key on my keyboard, so
I use Home.  

A window will open that looks like this:

 １記号入力  ２コード入力  ３部首入力  ４単語登録  ５環境舌逅跂闕

The one to select is 4, tangotoroku.

If you use kinput2 to manage your personal dictionary, you can still
get by with mule. 

You might want to try the key shortcut for mule.  Esc [ 2 8 ~

It is kind of hard to remember the first few times, but you get used to
it.
