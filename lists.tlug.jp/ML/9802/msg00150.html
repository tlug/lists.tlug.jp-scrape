東京 LINUX USERS GROUP 定例ミーティング
　

さる1998年2月14日(土曜日)、東京 LINUX USERS GROUP の定例
ミーティングが和気合い合いと行われました。以下はそのときの記録
です。これを書いている人間の記憶と知識があやふやなので、既に
日曜日にメイリングリストに投稿された、Craig Oda 氏の notes を
勝手に参考にさせて頂きました。さらに Ken Harada 氏に情報の
補足をしていただきました。ありがとうございました。

集合：12時30分、東京駅八重洲中央口
昼食会：中華料理店「たいこうろう」
昼食後の待ち合わせ場所：13時30分ごろ、東京駅地下構内
コーヒーショップ「スターバックスコーヒー」
ミーティング：HSCB James Capel 証券会社 (今回は地下一階)
　

***** 待ち合わせ *****

今回は自分は昼食会には参加しなかったのですが、前回よりさらに
増えて24人ものメンバーが中華料理「たいこうろう」に集まったそう
です。この段階ですでにビールが出ていたそうで、舌逅跂闕い顔をして会場
に来たメンバーも。

昼食の後で合流するメンバーは、スターバックスというコーヒー
ショップで待ち合わせます。Debian のパッケージをテーブルの上に置
いて、メンバーを待っていると、大きな LINUX とペンギンのロゴを
帽子に付けて歩いている不審な人物がやってきました。その他、新
しいメンバーがカップルでやって来ました。(バレンタインデーだったと
いうことで、そーゆうのもアリです。奥様や恋人を顧みず、終電まで
TLUG と飲んでいた人達は、見習いなさーい！)
　
　

***** ミーティング *****

1. 運営資金

実際のミーティングは、資金の確認から始まりました。TLUG 運営収
支に関する情報は、
http://www.twics.com/‾craig/tlug/finance/index.html
を参照してください。

2. 役員選出

役員は、資金の配分や運営方法などについて決定権を持ち、TLUG
の活動の方向づけを担当します。選ばれたのは以下の方々です。

Joe Marchak 氏
John Little 氏
Ken Harada 氏
Jim Schweizer 氏
　

3. 行事内容について

基本的に、発表や情報交換などを行う場として偶数月の第二土曜日
にミーティングを舌逅跂闕けます。普段のメイリングリストの上での交流から、
ゲストスピーカーや内容がしぼられます。

さらに、奇数月の第二水曜日には水曜ミーティングが舌逅跂闕けられます。
水曜ミーティングの内容は「飲む」と「食う」です。(つまりただの飲み会
ということです。もちろん、飲み食いしながら Linux Talk するのです。）
　

4. デモンストレーション

今回は４つほどデモンストレーションがありました。いまいち理解しき
れなかったものもあるのですがなかなか濃い内容でした。

◎ Alberto Tomita博士のGUI利用技術の研究内容発表。光源を二つ
とり、一台のカメラを使って光の反射方向から二次元／三次元の物体
の動きをとらえる研究を Linux を使ってされていて、それを応用する
と、視覚障害を持つために点字式ディスプレイを使用しているユーザ
の手の動きをマウスと同じように機能させることができるというお話し
だったと思います。
参照：http://www.crl.go.jp/st/st821/research/sikaku.html

# 以下、一緒に聞いていた Harada 氏の解舌逅跂闕です。
#
# 郵政省通信研究所との共同研究による、画像処理によるHuman
# Interfaceの研究でした。私もいまいちよくわかりませんでしたが、
# 特に「手」の動きをカメラで追って、画像処理するのをLinux
# でやりました、ということだと思います。応用としては障害を持つ方
# のための入力装置、ということですね。光学の数式はLinuxの会、
# という面からはわからなくてもしょうがないと思いました。カメラ1台
# で高さがわかる、というのは光学/数学的な知識がないと理解
# するのはむずいです。マウスと比べて格段にコストが高くなります
# が、手の動きでマウスカーソルをシミュレート、親指の角度で
# クリックというのはわかりやすかったかと思います。
# imasyにvolunteer working group というのがあって、難病患者さん
# を支援しているのですが、そちらに紹介してみるつもりです。

◎ Unicode エディタ Yudit。開発した Gaspar Sinai 氏が自らデモン
ストレーションを行いました。五か国語が一つの窓にきれいに表示
され、入力や編集、印刷もできました。英文に使用する普通の
postscriptプリンタで印刷できるということです。ハングルが分かる
メンバーがいれば、韓国語が舌逅跂闕しく表示入力できるかテストできた
ことでしょう。入手は、Sunsite の pub/incoming の下と、作者の
ホームページから、どうぞ。パッチも出るそうです。
http://www2.gol.com/users/gsinai/yudit-0.97.tar.gz
http://www2.gol.com/users/gsinai/patches/

◎ Pacific HiTech の Scott Stone 氏が　 TurboLinux Pro (standard
と professional 版)をTLUGに寄贈しました。早速インストールしてい
ました。私はインストールのデモンストレーションは見ませんでしたが、
Wnn6 と dp/Note を初めて使いました。普段は Canna で mule なの
です。慣れればさくさく使えるのかもしれませんが、dp/Note は私に
はちょっとむづかしかったです。

# コピーなどが一太纔瘢韭絎竢躡的なのはちと抵抗ありました、というメンバーも。
# 一太纔瘢韭絎竢躡　: 動作 -> 対象 -> 行き舌逅跂闕
# Word、Mule : 対象の選択 -> 動作 -> 行き舌逅跂闕　

◎ TLUG 初CDROM作舌逅跂闕。XCDroast を使って持ち寄ったプログラム
等を CD-R に焼きました。この作業は同じハードウェア環境で
Windows NT で行うと、Linux よりも時間がかかるそうです。ゆえに
Linux はこんなところでも優秀なんだということになりました。
コレクターズアイテムになることが予想される(？)TLUG の初
CDROM はTim Meggs 氏が抽選で獲得しました。

# ちなみに焼き込んだソフト/データは
# - TLUG mailing listの全アーカイブ
# - TLUG web pageの全コピー
# - Yuditのソースとバイナリー
# - SXLT_0.5a (Sparc-Linux Xterminal package)
# だったと思います。約70MBでした。
# 今後はいろんなソフトを集めてTLUGオリジナルを作っていくことに
# なるでしょう。

5. 抽選会

ナイスなLinuxグッズがあたる、恒例のお楽しみ抽選会です。今回の
人気は Red Hat の Tシャツ（白と黒3着づつ）でした。その他、
ラッキーな人々に Sparc や マウスパッド、キーボードが次々あたり
ました。
　

**** これからの展望 ****
これからも LINUX 愛好者のコミュニティにさらに貢献するため、さま
ざまな企画がもちあがっています。

◎ TLUG SERVER
舌逅跂闕線を引いて TLUG 舌逅跂闕のサーバを置こうという目的です。CD
ROM の内容を格納する場にするなどの利用方法が提案されていま
す。この件に関しては、Pacific HiTech の Scott Stone 氏から、PHT
がホスティングしてくれるかもしれないというお話がありました。

◎ CDROM
TLUG 独自のプログラムや資料などを CD-R に焼き、特製 CDROM
を配布する案がでています。Joe Marchak 氏が主になって行ってい
ます。FTP サイトを確保して、格納する計画です。

◎ TLUG LIBRARY
書舌逅跂闕やソフトウェアの貸し出しを行います。規則など詳細については
これから検討して決められていくと思います。現在 Joe Marchak 氏
が担当していますが、手伝ってくれる人を募集しています。

◎ The Perl Journal
The Perl Journal 日本語版を出版できるかどうか、TLUG 内の有志
で検討します。Perl を使うユーザは多く、今後も増えると思われます。
Joe Marchak 氏の案で、これからミーティングを重ねて可能舌逅跂闕を検討
する予定です。日本人側は今のところ加藤がお手伝いしますが、
我こそは適任！という方いましたら、申し出てください。初ミーティング
参加者は、2月28日、土曜日、13：30 に表参道交番前集合です。
Linux ユーザだけでなく、FreeBSD ユーザで Perl を使う方の協力も
呼びかけています。Chris Sekiya 氏が BSD関連に声をかけてくれる
そうです。
