>> 君塚＠Linux-Fairです。
>> 
>> Tokyo Linux Fair v0.5 への出展者を募集します。
>> 
>> 本イベントで舌逅跂闕けられるデモスペースにおいて、実演によるデモンスト
>> レーションをしたり、来場者や他の出展者とのオフラインのコミュニケ
>> ーションを通じて、自分のプロジェクトやフリーウェアのプロモーショ
>> ンをはかることを、ここでは総称的に出展といいます。出展者はプロジ
>> ェクト単位／個人単位のいずれでも歓迎です。
>> 
>> 現在、デモ機として５台のPCの用意があり、これら全台に一律に全出展
>> 者のソフトを入れ、どのマシンでも機能的に同じデモができるようにし
>> たいと思います。ただし、ディストリビューションレベルでの出展／デ
>> モに関しては、この限りではありません。なお諸事情により、現時点で
>> は全台スタンドアローン環境を前提としています。
>> 
>> 出展にあたり、デモスペースで実際にどんなことをするか、どういった
>> ものを用意するかといったことは、全面的に各々の出展者に委ねます。
>> こちらでやっておくこととしては、デモ機を用意し、せいぜいウィンド
>> ウマネージャや壁紙にバリエーションをつける程度にとどめます。
>> 基本的に売買行為でない限り、どのような企画・コーディネートでも自
>> 由です。特に東京圏のLinuxerの皆さんは、この機会を利用されてみては
>> いかがでしょうか。
>> 
>> 出展が決まりましたら、実行委員会メーリングリストに登録の上、同ML
>> 上で出展の申し出をして下さい。デモスペースへの出展者リストに加え
>> させていただきます。また、出展者へのアナウンスなどを同ML上で行な
>> います。募集期間は今月いっぱいまでを予定してますが、それ以後の申
>> し出があった場合でも、できるだけ対応したいと思います。なお、既に
>> 出展を申し出ておられる方は、改めてこのアナウンスに応じる必要はあ
>> りません。
