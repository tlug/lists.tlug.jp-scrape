Guys,

     I knew I'll learn more about Linux just by attending the Sunday
meeting. You guys are simply incredible! You know lots of Linux stuff.

     Anyway, I made a list on what I've learned after attending my first
TLUG meeting. Please make any corrections/comments if you find some errors.
So, here goes nothing:

     1. The Linux kernel is multi-threaded starting from 1.3.x. But there's
        no clear indication of a performance improvement. (Meaning, there's
        no sense messing with it yet.)

     2. You can use MS-DOS on Linux with the help of DOSEMU. And there's a
        rumor that you can run Word 6.0 on WINE. (Cool!)

     3. Red Hat is simpler to install and upgrade than Slackware. (This is
        good news for average users like me.)

     4. With SAMBA(?), you can use the resources of a Windows-based network
        from a Linux box.

     5. You can track a satellite using Linux. ( This is simply awesome. I
        never thought Linux could be this powerful. )

     6. Even the experts don't agree on the best editor - Jim likes jed and
        Steve likes emacs.

     7. pkzip can split a binary file; so you can save big binary files into
        multiple small-capacity diskettes.

     By the way, thanks Jim for hosting the meeting. It was fun and educa-
tional. Otsukare sama deshita.

TFN,
George

P.S.
     Let's use the 1,000-yen change from the Spaghetti Factory to buy some 
soda next meeting.
