Hi,
I was going through the Debian-JP site when I found a nice
installer for Netscape Communicator v403.   It sets
the X11R6 locale/ja up so that the cut and paste feature
works with Japanese.  There is also a nice resource option
that sets all the menus to Japanese.   

I thought this was a pretty slick way to get the Japanese
cut and paste support to work with Netscape under Linux.

get it from:

ftp.debian.or.jp/pub/Linux/debian-jp/contrib-jp/binary-i386/netscape-ja_v403-3.deb

Here's the readme:  (I hope the iso-2022-jp is readable now)

The Debian Package netscape-ja
----------------------------

このパッケージは Netscape Communicator 4.03 （英語版）を無理矢理日
本語表示させるものです．

この日本語リソース（Netscape-ja*.ad）に関する質問は、Netscape
Communications 社には送らないでください．

X11R6.3 上で動作させると舌逅跂闕常に日本語が表示されない場合があります．
XLC_LOCALE.patch をお試しください．

    # cd /usr/X11R6/lib/X11/locale/ja
    # patch -s < /usr/doc/netscape-ja/INSTALL/XLC_LOCALE.patch


Description: Netscape communicator インストーラ・キット
 このパッケージは Netscape communicator のインストーラです．
 このインストーラを使ってインストールする前に，以下のファイルを /tmp 
 に置いてください．
    communicator-v403-export.x86-unknown-linux2.0.tar.gz
    navigator-v403-export.x86-unknown-linux2.0.tar.gz
 ただし， communicator または navigator のみを使うことも可能です．
