> I am new to this list. I have just put a question to the message board but as I have decided to join the mailing list, I would like to repeat it here. (sorry for this double message)
> 
> Can anybody give me any hints how to configure a japanese version of linux (in my case RedHat 6.2J) to support german special characters (umlauts) AND japanese input (kinput2). I can only get one of them working, depending on the LANG variable. If I define LANG as "de_DE", the japanese support doesn't work. If I define it "ja_JP" the german umlauts don't work for example in kedit (although I have selected "german" in the kde-International keybord support) and in cjk-lyx.
> 
Well, to make it possible we need wait for unicode to be standarized in linux.
If you want some apps running with a japanese environment and others with german environment, you can run them through the "env" command, like this:

env LANG=ja_JP gimp
then gimp will run with japanese environment whether the LANG variable was set to japanese or not. Of course, this change will be only to the application. all other applications you run will still use the original value.
