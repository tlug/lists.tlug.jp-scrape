I would like to comment once more on the usefulness of Jim Breen's
Dictionary Server.

It looks as if the help of Jim's Online Dictionary is even undervalued
(correct English word?) by Jim himself.

Input
間違ったファイルシステムタイプ、不舌逅跂闕なオプション
Output
間違う 【まちがう】 (v5u) to make a mistake; to be incorrect; to be
mistaken; EP
ファイルシステム filesystem; ED
タイプ (n) type; style; typing; EP
不舌逅跂闕 【ふせい】 (adj-na,n) injustice; unfairness; iniquity; impropriety;
irregularity; dishonesty; illegality; EP
オプション (n) option; EP

My small changes:
to make a mistake; to be incorrect; to be mistaken => wrong
filesystem
illegality => illegal
option

with these words and a suitable search engine you will probably find some
help;
for such kind of problems the usenet groups are first choice
searchable at http://groups.google.com/
