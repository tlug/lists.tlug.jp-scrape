>> You should install freewnn and kinput2.
>> Then do 'export LC_ALL=ja_JP" in your shell before running the desired
>> application.  You should be able to input japanese into netscape, mozilla
>> (some builds don't work for me), gnome, kde, kterm, and so on.
>> You should also read the manpages, some relevant howtos and the tlug
>> archieves if things don't work.

An odd thing. I was experimenting with this. I set LC_ALL, LANG and
LANGUAGE (for good measure) to ja_JP, then started Nestscape from within
that shell. Instead of it starting, I got:

<quote>
netscape: プログラム バ・踉札献腑鵑ですが、リソ・踉札垢離弌
ジョンは 4.72 です。

        これは不適舌逅跂闕な `Netscape' ファイルが app-defaults ディレクトリに
        インスト・踉札襪気譴討い襪燭瓩任后０焚の環境変数とそれが┐
	ディレクトリを確認してください

        $XAPPLRESDIR
        $XFILESEARCHPATH
        $XUSERFILESEARCHPATH

        また、ホ・踉札ぢディレクトリや、
 /usr/lib などにある `app-defaults' 
        ディレクトリ内のこのファイルも確認してださい。
</quote>

Sure enough, I looked in /usr/X11R6/lib/X11/ja/app-defaults/Netscape
and the file in there says:

! Netscape.ad --- app-defaults file for Netscape 4.72. 

I think this came with RH6.2. I replaced the 4.72 with 4.76 some time ago, 
but obviously that installation didn't redo the app-defaults file.

Any suggestions? Do I get another file from somewhere? Can I just
modify the existing one?
