Hullo there Jim!

I would recommend mplayer. You need to have codecs like ffanim,
opendivx, or some special Windows codecs, plus DVD libraries if you need
them, installed BEFORE you ./configure && make mplayer. The output from
configure will tell you what's enabled and what's not. Also, ./configure
--help will tell you how to enable and disable certain codecs and
performance-enhancing features. --disable-runtime-cpu-detection for
instance.

If you have Gentoo Linux, you can USE="fbcon sdl directfb opengl sse dvd
mmx alsa nls cjk opengl aalib... you get the picture" emerge mplayer and
get it all installed for you.
