On Sep 8, 11:03pm, Craig Oda wrote:
} Subject: Re: tlug: kterm/kinput2 conversion

Craig Oda suggested:

>> This is my .Xresources
>> 
>> kterm*kanjiMode: euc
>> kterm*visualBell: true
>> kterm*VT100*Translations: #override \
>>         Shift<Key>space: begin-conversion(_JAPANESE_CONVERSION)
>> kterm*preeditType:   Separate,OverTheSpot,OffTheSpot,Root 
>> kterm*inputMethod: kinput2 

which I put in ~/.Xresources

I then did

>>  $ xrdb -merge ~/.Xresources

And it worked! kterm now runs an "over the spot" conversion.

So the problem was missing resources.

Many thanks. Craig wins the "first with the correct answer prize.
