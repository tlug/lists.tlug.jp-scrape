On 08 Feb 2001 13:14:04 +0100
Mike Fabian <mfabian@example.com> was chatting the other day about l and said:

> David Eduardo Gomez Noguera <davidgn@example.com> writes:
> 
> [...]
> 
> > and how do you use radical input with canna? i currently use kinput2.
> 
> After going into Japanese input mode (usually shift-space with
> kinput2), hit the "home" key[1].  A window pops up with the contents:
> 
>     １記号入力  ２コード入力  ３部首入力  ４単語登録  ５環境舌逅跂闕
> 
> Select 「部首入力」 (by typing "3", or change the selected item
> with the arrow keys or by using C-f, C-b and press RET).
> 
> A window showing a list of radicals pops up. Select one
> (by clicking with the mouse or by moving the selected item with
> the arrow keys or C-f, C-b, C-n, C-p and type return).
> 
> For example, select 辷（しんよう）. A new window with characters
> containing this radical pops up. Among them is also 邉.  Select the
> character you want.
> 
> You can always go back to the previous pop-up-window by typing C-g.
> 
> 
> Footnotes:
> [1] The manual of Canna says it is the "help" key, but it
> seems to be the "home" key by default.
> 

Thank you a lot. 
this seems not to work on gnome terminal, but it works on kterm.

and it has some other strange behavior in other programs.
maybe it has something to do with the window manager?
